## Technology Stack
- **Frontend**: Streamlit
- **Orchestration**: LangGraph
- **LLMs**: Configurable via `src/utils/config.py` (DeepSeek, OpenAI, etc.)
- **Search**: Tavily / Serper / Custom
- **State**: Pydantic models (`src/models.py`)
- **Async**: `asyncio`, `aiohttp`
